# WBGT 表示追加計画

## 目的

OLED の表示を再配置し、DI を気圧の直下へ小さい文字で 1 行表示する。空いた左下の指標欄には、気温・相対湿度・気圧から算出する屋内 WBGT を表示する。

## フェーズ

### Phase 1: 要件と設計の確定

- 要件、仕様、設計、ADR を作成する。
- OLED 座標、文字サイズ、WBGT の計算式および異常値時の扱いを確定する。

状態: 完了

### Phase 2: 実装

- `ComfortMetrics` に WBGT を追加する。
- 飽和水蒸気圧、湿球温度、WBGT の計算を追加する。
- DI と WBGT の OLED 表示を指定位置へ再配置する。
- `README.md` の機能説明を実装と同期する。

状態: 完了

### Phase 3: 検証

- `arduino-cli compile --fqbn arduino:esp32:nano_nora .` を実行する。
- 実機で温湿度・気圧の取得、DI と WBGT の表示位置、Wi-Fi 再接続を確認する。

状態: コンパイル完了。実機確認は未実施。

## 成功基準

- OLED で DI が `DI 76.0` のように、気圧の直下に文字サイズ 1 で 1 行表示される。
- 従来の DI 表示位置に WBGT が表示される。
- WBGT は気温・相対湿度・気圧を用いる湿球温度ベースの式で算出される。
- コンパイルが成功し、実機表示が仕様どおりである。
