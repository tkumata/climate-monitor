# ADR-002: OLED を左右 2 列の情報密度優先レイアウトにする

## 状態

採用

## 背景

気温、WBGT、DI は閲覧時に優先度が高く大きな文字が必要である。一方、気圧、相対湿度、絶対湿度、VPD も同一画面で確認できる必要がある。

## 決定

128×64 OLED を左右 2 列に分ける。左列には気温、WBGT、DI の値を文字サイズ 2 で表示し、右列には気圧、相対湿度、絶対湿度、VPD を文字サイズ 1 で表示する。

## 結果

- 主要な 3 指標を見やすく表示できる。
- 4 つの補助指標を画面遷移なしで確認できる。
- 計算、通信、保存形式は変更しない。
