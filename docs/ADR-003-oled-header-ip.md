# ADR-003: OLED ヘッダーに STA 側 IP アドレスを表示する

## 状態

採用

## 背景

設定ページの接続先を OLED 上で確認できるようにしつつ、固定の `Meteorology Info` タイトルを表示領域から除く必要がある。IPv4 アドレスの文字数によらず 128px 幅に収める必要がある。

## 決定

`Meteorology Info` と `http://` は削除する。STA 接続中は `WiFi.localIP()` の IP アドレスだけを最上段に描画する。STA 未接続時は表示可能な STA 側アドレスがないため IP アドレスを描画しない。日付・時刻と計測値の描画位置は変更しない。

## 結果

- 表示中の IP アドレスを使って STA 側から設定ページを開ける。
- 最大 15 文字の IPv4 アドレスを表示できる。
- 計算、通信、保存形式は変更しない。
