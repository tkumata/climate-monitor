# WBGT 表示追加要件

## 機能要件

1. OLED の DI は、文字サイズを既存の 2 から 1 へ下げる。
2. DI は `DI 76.0` の形式で同一行に表示する。
3. DI は気圧表示の直下に配置する。
4. 従来の DI 表示位置には WBGT を表示する。
5. WBGT は気温、相対湿度、気圧を用いて算出する。
6. WBGT は、飽和水蒸気圧と二分法で求めた湿球温度から `0.7 × 湿球温度 + 0.3 × 気温` として求める。
7. 計算入力が有限値でない、または気圧が 0 以下の場合、WBGT は無効値として扱う。

## 非機能要件

- 既存の OLED 解像度 128×64 と描画構成を維持する。
- Wi-Fi 設定、EEPROM レイアウト、エッジサーバーへの送信ペイロードは変更しない。
- 動的メモリ確保を反復描画経路へ追加しない。
