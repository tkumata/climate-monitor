# OLED ヘッダー IP アドレス表示要件

## 機能要件

1. OLED の `Meteorology Info` タイトルを表示しない。
2. STA 接続中は、OLED 最上段に STA 側 IP アドレスを表示する。
3. STA 側 IP アドレスは `WiFi.localIP()` から取得する。
4. STA 未接続時は IP アドレスを表示しない。
5. 既存の日時および計測値の表示、WBGT、DI、絶対湿度、VPD の計算式と異常値の扱いは変更しない。

## 非機能要件

- 既存の OLED 解像度 128×64、文字サイズ、および描画構成を維持する。
- Wi-Fi 設定、EEPROM レイアウト、エッジサーバーへの送信ペイロードは変更しない。
- 動的メモリ確保を反復描画経路へ追加しない。
