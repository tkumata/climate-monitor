# ADR-001: 気圧を含む湿球温度ベースの屋内 WBGT を採用する

## 状態

廃止（ADR-008 で置換）

## 背景

WBGT 表示には、気温と相対湿度に加え、BME280 が取得済みの気圧を利用する必要がある。

## 決定

飽和水蒸気圧、実際の水蒸気圧、乾湿計係数を使い、二分法で湿球温度を求める。WBGT はその湿球温度と乾球温度の重み付き和として計算する。

## 結果

- 気圧を WBGT 計算に反映できる。
- 固定 50 回の反復で実行時間とメモリ使用量を一定にできる。
- 気圧が無効な場合は有効な WBGT を算出できないため、NaN を返す。

## 置換

反復計算をなくし、Stull の湿球温度近似式へ統一するため ADR-008 で置換した。
